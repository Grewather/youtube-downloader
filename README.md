# youtube-downloader
simple youtube downloader made in node.js

![image](https://user-images.githubusercontent.com/69985703/174576586-89c58416-44a4-4f1c-9712-37caa9e3cdf5.png)

## Stack
* nodejs
* nodegui
* ytdl

## Installation
Install this packages to run project
```
npm i @nodegui/nodegui
npm i @nodegui/qode  
npm i ytdl-core 
```

## Usage

Run project using "npx qode downloader.js" command
